# Quick Start Guide

Welcome to the Solar Orbiter Flare Link tool quick start guide!

Navigate this web application by using the menu at the left. The main functionality is located under [SEP Event Analysis](/). 

## Core Functionality

The app is designed to help establish a link between solar flares observed by STIX and _in situ_ energetic particle events measured by EPD on Solar Orbiter. A number of criteria are automatically assessed for a given event

<div style="background-color: rgba(33, 150, 243, 0.1); border-left: 4px solid #2196F3; padding: 15px; margin: 10px 0; border-radius: 4px;">
<strong>Key Criteria:</strong>
<ul style="margin: 10px 0 0 0;">
<li>If there exists a magnetic connection between the flare footpoints and Solar Orbiter</li>
<li>Given the onset of the flare, check whether the travel time of particles along the Parker Spiral matches with energetic particle events observed by EPD</li>
</ul>
</div>

For a detailed description of the criteria, please refer to the [documentation](/documentation#Criteria).

The information is presented in four different ways

1. **Flare Overview** giving a tabulated overview of events which satisfy the selected criteria in a given time range
2. **Flare Detail View** giving a detailed view of a single flare and possible associated events
3. **Flare Connectivity Summary** giving a graphical overview of events which satisfy the selected criteria in a given time range
4. **EPD Sensor Summary** giving an overview over all flares split by EPD sensors in a given time range
