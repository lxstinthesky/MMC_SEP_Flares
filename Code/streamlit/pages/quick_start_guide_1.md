### Flare Overview

The flare overview provides a tabulated summary over all flares in the selected time range, matching the selected criteria. By clicking on the column header, the table can be sorted in ascending or descending order. The columns can be dragged and rearranged arbitrarily.

The most important adjustments are made with the three dropdown boxes. Different sensors from the EPD instrument like EPT-SUN or STEP can be selected here.
