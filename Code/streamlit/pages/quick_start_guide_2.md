### Flare Detail View

The flare detail view is designed for a more in depth analysis of a single flare and offers insights into detailed magnetic connectivity information, the energetics of an SEP and STIX lightcurves. To active it, select a flare from the [Flare Overview](/quick_start_guide#flare-overview) by checking the box on the far left of the table. Only one flare can be selected at a time.
